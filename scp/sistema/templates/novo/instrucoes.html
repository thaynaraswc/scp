Novo Usuário

Como o acesso ao sistema é totalmente restrito, o cadastro de novo usuário precisa ser realizado pelo administrado do sistema. Assim, ele possa também irá dar permissões de acesso e organizar toda a sua área de usuário.

Envie um email com os dados abaixo para o email do administrador (contato@adm.com)

  • Nome completo
  • CPF
  • Código Funcional
  • Data de nascimento
  • Cargo
  • Setor
  • Número de contato (Ramal)
  • Localidade que está lotado(a)

    • Completando cadastro

    Como o acesso ao sistema é totalmente restrito e o cadastro de novo usuário é realizado pelo administrado do sistema, os dados como endereço, telefones extras de contato, e demais dados que o administrador não sabe precisam ser preenchidos. Para isso, basta seguir o tutorial abaixo para compreender melhor. Lembramos que é de extrema importancia que os dados estejam todos preenchidos.
